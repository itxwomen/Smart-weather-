# Smart-weather-
My first Python project: A smart tool that will help you choose what to wear according to the weather.
# About the project
The program gives personalized recommendations on clothing depending on the weather. 
The project was created to tech the basics of Python.
# Opportunities
- Annalizes the current temperature 
- Takes into account the presence of rain
- Processes the wrong answer
- Works in a loop until the correct input
# Utilization 
1. Start the program
2. Enter how many degrees today
3. Indicate if it's raining (да/нет)
4. Get recommendations on clothes
